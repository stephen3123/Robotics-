ARDUINO – 2 BUTTONS

Overview

Reaction Speed Tester for Left and Right Hand

This project allows you to test the reaction speed of your left and right hand using Arduino, a display, and two buttons.

The program records the time it takes to press each button, compares results, and displays the value in milliseconds. After several attempts, the average value is shown.
